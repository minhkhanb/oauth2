
## logonoff

This is the code listing for a tutorial about using OAuth2 in
client-side JavaScript apps. You can read it at
[tests4geeks](https://tests4geeks.com/tutorials/oauth2-javascript-tutorial/).

To run the sample, download this directory into `logonoff` and run:

```bash

virtualenv venv
. venv/bin/activate
pip install -r requirements.txt

python app.py

```

You'll need python and virtualenv installed.
